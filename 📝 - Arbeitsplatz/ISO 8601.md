Der Standard **ISO 8601** legt ein Datumsformat fest. Typischerweise ist dies `YYYY-MM-DD`. Dieses Format hat gegen√ºber den anderen beiden typischen Formaten `DD-MM-YYYY` und `MM-DD-YYYY` den Vorteil, das es von Text-Sortier-Algorithmen chronologisch sortiert wird.

Es ist damit anderen Formaten immer vorzuziehen.

Wichtige von **ISO 8601** definierten Formate:

| Typ                   | Format            | Beispiel |
| --------------------- | ----------------- | -------- |
| Datum mit Jahr        | `YYYY-MM-DD`      |  2021-01-23        |
| Datum ohne Jahr       | `--MM-DD`         |      --01-23    |
| Datum mit Zeit in UTC | `YYYY-MM-DD<T>HH:mm:ss<+><HH:mm(Zeitzonenunterschied)>` |        2021-01-23T15:34:31+00:00  |

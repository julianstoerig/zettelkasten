Die **Quantenzahlen** beschreiben die Eigenschaften eines Elektrons eines Atoms, die zusammen das Orbital und den Platz im Orbital bestimmen. Die **Quantenzahlen** eines speziellen Elektrons lassen sich aus der [[Elektronenkonfiguration]] dessen [[Atom|Atoms]] entnehmen. Es gibt vier Quantenzahlen:

|                               | Eigenschaft des Elektrons, die sie beschreibt |
| ----------------------------- | --------------------------------------------- |
| [[Hauptquantenzahl n]]        | Schale                                        |
| [[Nebenquantenzahl l]]        | Unterschale                                   |
| [[magnetische Quantenzahl m]] | Orbital                                       |
| [[Spinquantenzahl s]]         | Platz im Orbital (Spin)                    |
